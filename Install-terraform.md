## How to download Terraform in Linux System ?
1. First Login to the Linux System.
2. Then install the terraform
- Go to terraform.io website 
- then go to install section
- Then select the operating system **Linux**
```sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://rpm.releases.hashicorp.com/RHEL/hashicorp.repo
sudo yum -y install terraform```
